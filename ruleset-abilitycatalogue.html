Feature & Flaw Tier Costs

Centralized cost tables. Every Feature and Flaw in every Builder references a tier name from these tables — change a tier's value here and every Feature/Flaw at that tier re-prices automatically across the entire Catalogue.

Default Features

Features that auto-populate every Builder in this catalogue. On each Builder, the GM can suppress an individual default or override its name/description/tier/stackable.

Default Flaws

Flaws that auto-populate every Builder in this catalogue. Same override rules as Default Features.
