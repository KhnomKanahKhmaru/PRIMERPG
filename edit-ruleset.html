Starting XP & Power Levels
How much XP new characters start with, and how expensive AP is to purchase at each Power Level.
Each level has a label and a cost in XP to purchase 1 AP. One must be marked as default (new characters start at this level).
Stats
Define the stats available to characters, their scaling caps, XP costs, STATMOD per level, and flavor labels.
Each stat has a short code (e.g. STR), a name, and a description shown on hover.
Max Purchasable is the highest level a player can start a stat at with XP. Absolute Max is the hard ceiling even through play.
XP cost only applies up to Max Purchasable. STATMOD and Label apply across the full scale.
Size
Size is a separate scale from regular stats. Define the tiers available and what each one is called. Size has no STATMOD — it's purely a scale + label that affects other math elsewhere in the game.
Each tier has a numeric level, a label, and an optional XP cost for starting characters to choose it.
Skill Costs
XP cost per level for each skill tier. The max level determines how high players can push a skill.
